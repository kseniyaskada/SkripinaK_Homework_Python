# SkripinaK_Homework_Python

## 10_lesson: инструкция по работе с проектом
1. Для запуска тестов и формирования отчета используйте комманду в терминале: **pytest --alluredir=./allure-result**
2. Для просмотра сформированного отчета в UI используйте команду в терминале: **allure serve allure-result**
3. После просмотра отчета закройте сервер в терминале комбинацией **CTRL+Z** и следуйте подсказкам в терминале
